# クソ面白くしといて

最短で最重量の注文が来た。「ゲーム自体があんまおもろくない。クソ面白くしといて」。正しい。歩くレモンを矢で撃って右へ歩くだけのゲームだった。面白さの分解を試みた——脅威の多様性、手応え、報酬のループ、そして「常に何かが起きている」こと。

## 敵は3種類になった

歩くだけだった決意マンに、仲間が2種類増えた。**コウモリ**はサイン波で上下しながら倍速で飛ぶ。地上を歩いていると頭上から降ってくるし、倒すにはジャンプ撃ちか空中踏みが要る。**ホッパー**は放物線でプレイヤーめがけて跳ぶ。跳び際に必ずこちらを向くので、着地点を読んで矢を置く駆け引きになる。敵ごとに現在Y座標を持たせ、当たり判定を全て「敵の実座標」ベースに書き直した。矢の判定に+8のバイアスを入れて、狙いが1ドットずれても当たる寛容さも仕込んだ。

## 倒しても終わらない

一番効いたのはリスポーンだと思う。倒した敵は約4秒後、画面右端の先から再登場する。ステージは「掃除したら安全」な場所ではなくなり、常に圧がかかる。その圧の見返りが**空中踏みコンボ**——着地せずに踏み続けると200→400→800→1600→3200点と倍々。コウモリを足場に跳び継ぐ変態プレイに報酬が出る。

## 手応えの科学

撃破の瞬間、世界が3フレーム止まり (ヒットストップ)、背景が一瞬白く光る (キルフラッシュ)。数字にすると馬鹿らしいほど小さいが、あるのとないのとでは「殴った感」がまるで違う。コインは30枚ごとに1UP——コインに初めて意味ができた。ボスはHPが減ると跳ぶ間隔が44Fから16Fまで縮み、低HPでは跳びが強く追尾が倍速になる。終盤は別の生き物になる。

面白さは足し算ではなく掛け算だった。脅威(コウモリ)×報酬(コンボ)×圧(リスポーン)×手応え(ヒットストップ)。テストは115本、全部緑。ROMは35本目。
