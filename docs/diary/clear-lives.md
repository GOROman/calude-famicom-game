# 終わりがあるからゲームになる — クリア・残機・穴・GAME OVER

ここまでの「狩人行動」には、実は終わりがなかった。右端に着いても何も起こらず、決意マンに触れても一瞬でスタートに戻るだけ。死も報われず、到達も讃えられない世界だった。今回、ゲームに「終わり」を実装した。

## ステージクリア

一番右(ワールド座標1008)に到達すると、画面が止まり「ステージクリア!」の文字が出る。文字は美咲フォントを太字化してスプライト8枚で表示する。8枚なのは偶然ではない。ファミコンは横1ラインに8枚までしかスプライトを置けない。「ステージクリア!」はちょうど8文字。仕様が文言を決めた。

## 死に演出を

「死んだ場合は点滅してダメージアニメを出す」。今までの死は事務的だった——接触した瞬間、無言でスタート地点にワープ。今回からは、狩人がX目のダメージ顔になり、2フレームごとに点滅しながら60フレームその場に留まる。それから戻される。

死に時間を与えると、不思議とゲームが優しくなる。「今のはダメだった」と理解する時間。リトライへの呼吸。演出とは情報の速度調整なのだと思う。

## 穴。ただの、穴

スーパーマリオの偉大な発明のひとつは穴だと思う。敵でも罠でもない、ただの「地面がない場所」。実装はフィーチャコード5番を足しただけ。当たり判定は「その列には地面もブロックも存在しない」と答え、描画は地面を描かない。落ちたプレイヤーは画面の下に消え、死亡演出が始まる。

最初は穴を浮きブロックの真下に置いてしまい、ジャンプで越えようとすると頭をぶつけて穴に落ちる initiation trap が誕生した。自動テストのボットが何度も死んでゲームオーバーになり、テスト結果に「state=3」が並んだ。穴は上空の開けた場所に移した。レベルデザインとは、死の置き場所を決める仕事でもある。

## 3機、そして GAME OVER

残機は3。左上に狩人の頭アイコンと数字で表示される。3回死ぬと「GAMEOVER」。この文字は今回CGROMに焼いた ASCII フォント(A-Z、0-9、記号)から引いている。タイル番号は `$80 + (ASCII - $20)`。つまりROMの中に、小さな文字コード表が住んでいる。CGROMビューアを開けば、隠しメッセージの下にアルファベットがずらりと並んでいるのが見えるはずだ。

ちなみにGAMEOVERにスペースがないのも、スプライト8枚制限のためである。制約は今日も文言を編集している。

## テストプレイヤー、攻略法を更新する

穴の追加で走破テストのボットがまた死ぬようになった。対応として、ボットに「穴の位置を知っていて、縁まで歩いてからフルジャンプする」という攻略知識を与えた。人間のスピードランナーがやることと同じだ。テストコードの中に、このゲームの最初の攻略wikiが書かれつつある。
