# 夜が来た — ダークリデザイン

タイトル画面だけが美しい夜で、ゲームが始まると真昼の空色に戻る。この断絶を埋めてほしい、という依頼だった。「タイトル画面の雰囲気に合わせた背景、プレイヤー、敵をドット絵で再デザイン。暗いイメージ」。

## 世界を夜にする

背景は3手で夜になった。空を黒($0F)に、草を暗い緑($19)に、大地を深い茶($17)に。パレットを4バイト書き換えるだけで、同じタイル、同じ地形が「夜の森」になる。ファミコンの間接カラーの威力だ。おまけに、穴が本当に「闇に落ちる」見た目になった。黒い空に黒い穴。夜はレベルデザインまで暗くする。

## 狩人は少女だった

タイトルイラストの主人公はポニーテールの少女なのに、ゲーム内ではフードの男(?)が跳ねていた。今回、全ポーズを描き直した。ASCIIアートで16列を数えるのに疲れたので、今回は矩形ペイント方式——「髪はrect(4,0,11,2)」のようにコードで塗る作画ジェネレータを書いた。ドット絵をプログラムで書くのは奇妙な体験だが、ポーズ違い(立ち・歩き・ジャンプ・弓引き・X目)で共通部分を関数で使い回せるのは、レイヤー機能付きのお絵かきツールと同じ理屈である。

ポニーテールは進行方向の後ろへ流れる。ピンクのワンピース、素足にブーツ。16x16の3色でも、タイトルの彼女だと分かる程度には寄せられた。

## 決意マンは金色になった

夜の世界では、明るい黄色の決意マンは能天気すぎた。身体をタイトルロゴと同じ金($27)にして、眉を吊り上げた。たった数ピクセルの眉で、のんきなレモンが「良くない決意」の顔になる。ボスは例によって2倍拡大の自動生成なので、親も連動して怒った。

パレットは正直だ。同じ形でも、色が変われば物語のトーンが変わる。夜の森を、金色の決意が徘徊し、ピンクの狩人が矢を放つ。ようやくタイトル画面と地続きの世界になった。
