# 決意の門 — ボスとエンディング

物語には結末が要る。決意マンの世界を旅してきた狩人カリュードに、ようやく倒すべき「本体」を用意した。1-4の最深部、柱の門の先のアリーナに鎮座する、32x32の大決意マンである。

## 大きいことは、それだけで怖い

ボスのドット絵は通常の決意マンを2倍に拡大しただけだ。それなのに、画面で対峙すると妙な圧がある。16スプライト(4x4)で組んだ巨体が、40フレームおきにこちらへ跳びかかってくる。HPは8。矢で1、勇気の踏みつけで2削れる。被弾すると20フレーム点滅して無敵になる——連射で溶けないボスは、それだけでボスらしい。

ボスが生きている間はステージクリアがゲートされる。門の先で決着をつけない限り、この世界は終わらない。テスト実装中に面白い光景があった。ボス戦のスクリーンショットに、通常サイズの決意マンが親子のようにボスの前を横切っていたのだ。修正しようかと思ったが、大小の決意マンが並ぶ画はむしろ良かったのでそのままにしてある。

## OAMは掃除されない

エンディングを実装したら、黒画面に旧ステージのスプライトが亡霊のように残った。原因は状態遷移の隙間。演出タイマーが切れて `show_ending` がスプライトを全消しした直後、メインループが「演出中はそのあと描画もする」という古い前提のまま1フレームだけHUDやプレイヤーを描き戻していた。ファミコンのスプライトは誰かが消すまで居座る。1フレームの描き込みが、永遠の残像になる。遷移フレームの描画をスキップして解決した。

## フェードインは音量だけではない

「タイトルBGMはフェードイン」。ファミコンには音量レジスタしかないので、素朴にやるなら全チャンネルの音量を徐々に上げる。でも実際の曲のフェードインは、そうじゃないことが多い——楽器が順番に入ってくるのだ。

そこで音量キャップを0から15へ2秒かけて上げつつ、**ベースはキャップ4から、ドラムはキャップ8から**参加させた。静かにメロディが立ち上がり、うねる303が加わり、最後に808が四つ打ちを刻み始める。RMSで測ると0.24→0.48。数字はただの倍音だが、聴感上は「幕が上がる」感じになる。

## 脚は大事

タイトル絵の少女の脚が、山脈の帯のところで欠けているという指摘があった。原因はパレット割当。脚と山が同居する16x16ブロックで、ピクセル数の多い山側の色が勝ち、肌色が追い出されていた。対策は「肌色系の誤差を3倍重くする」saliency(顕著性)の導入と、キャラクター領域のタイルをマージ保護すること。人間の目は人物の肌から見る。アルゴリズムにもそれを教えてやる必要がある。

これでゲームは 起(タイトル)・承(1-1〜1-3)・転(ボス)・結(エンディング)が揃った。THE END の7文字を、自分のフォントで表示できる日が来るとは。
